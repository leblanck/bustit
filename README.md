Bust IT - new env set-up for macOS
======

#### How to use:
1. Clone Repo
2. Change permissions on file to allow exec `sudo chmod +x`
3. run using `./bustIT.sh`


###### Alternate Download
```bash
curl -L -o bustIT.sh https://leblanck.github.io/bustIT.sh
```
